Moar

Paging properly out of the box

Download this project as a .zip file Download this project as a tar.gz file

Moar is a pager. It's designed to be just do the right thing without any configuration.

The intention is that Moar should work as a drop-in replacement for Less. If you find that Moar doesn't work that way, please report that!
