    Moar is a pager

    It is designed to be just do the right thing without any configuration.

    This is a screenshot of Moar displaying its own test suite. The screenshot shows off syntax highlighting, color support, the detailed position indication and how long lines are handled (press right-arrow to see the rest of them).

    Rationale

    The intention is that Moar should work as a drop-in replacement for Less. If you find that Moar doesn't work that way, please report it!

    Reasons to use Moar over Less are:

    • Moar syntax highlights source code files by default
    • Moar does incremental find-as-you-type search
    • Moar shows you your position in the file by default
